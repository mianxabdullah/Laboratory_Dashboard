# 🏥 Laboratory Operations Analytics Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard was developed to monitor and optimize laboratory operations by providing real-time visibility into the complete clinical sample lifecycle—from patient requisition intake to final report validation.

The dashboard enables laboratory managers and operational teams to track throughput, monitor turnaround times (TAT), identify workflow bottlenecks, and improve resource allocation through data-driven decision-making.

---

## 🎯 Business Objective

Laboratories process thousands of samples daily, making operational visibility critical for maintaining efficiency and ensuring timely patient care.

This dashboard was designed to:

* Monitor sample progression across laboratory stages
* Track operational turnaround times (TAT)
* Identify process bottlenecks
* Analyze specimen volume trends
* Monitor demographic distributions
* Provide real-time operational status updates

---

## 📊 Dashboard Features

### Executive Metrics

* Total Patient Records Processed
* Average Turnaround Time (TAT)
* Collection Volume Tracking
* Processing & Validation Performance
* Real-Time Operational Status Monitoring

---

### Sample Workflow Pipeline

Interactive pipeline tracking:

Requisition Added ➡️ Sample Collection ➡️ Processing ➡️ Validation

Users can quickly identify where samples are accumulating and monitor conversion rates between workflow stages.

---

### Turnaround Time Analysis

Key performance indicators include:

* Average Days from Added to Processing
* Average Days from Processing to Validation
* Overall Validation Cycle Performance

These metrics help laboratory managers monitor efficiency and improve service delivery.

---

### Specimen Analysis

The dashboard provides visibility into specimen distribution and workload drivers.

Sample categories include:

* Buccal Swab
* NP Swab
* Saliva Samples
* Other Laboratory Specimens

This analysis helps identify high-volume sample types and resource requirements.

---

### Demographic Insights

Patient population analysis includes:

* Age Group Distribution
* Gender Distribution
* Volume Trends by Demographic Segment

These insights support reporting and population-level analysis.

---

### Operational Status Monitoring

Real-time tracking of sample statuses:

* Ready for Extraction
* Ready for Processing
* Ready for Interpretation
* Processed Samples
* Validated Samples

This allows teams to proactively manage workload and prioritize tasks.
